================================================================================
  OBSIDIAN + MCP PRODUCTIVITY SETUP GUIDE
================================================================================

Hey! This guide will get you set up with Obsidian for daily note-taking and
an MCP server that lets Claude pull your outstanding tasks automatically.
Takes about 15 minutes.


================================================================================
  STEP 1: INSTALL OBSIDIAN
================================================================================

  macOS (Homebrew):
    brew install --cask obsidian

  macOS (Manual):
    Download from https://obsidian.md/download and drag to /Applications.

  Verify it installed:
    open -a Obsidian


================================================================================
  STEP 2: CREATE YOUR VAULT
================================================================================

You have two options: local-only or iCloud-synced.

  OPTION A — iCloud (Recommended: syncs to iPhone/iPad automatically)
  ---------------------------------------------------------------
  Your vault will live in iCloud Drive so it syncs across all Apple devices.

  1. Open Finder and make sure iCloud Drive is enabled:
       System Settings > Apple Account > iCloud > iCloud Drive (ON)

  2. Create the vault folder:
       mkdir -p ~/Library/Mobile\ Documents/iCloud~md~obsidian/Documents/Notes

  3. Open Obsidian > "Open folder as vault" > navigate to:
       ~/Library/Mobile Documents/iCloud~md~obsidian/Documents/Notes

     Tip: In the Finder dialog, press Cmd+Shift+G and paste:
       ~/Library/Mobile Documents/iCloud~md~obsidian/Documents/Notes

  OPTION B — Local only
  ---------------------------------------------------------------
  1. Pick any folder you like, e.g.:
       mkdir -p ~/Documents/ObsidianVault

  2. Open Obsidian > "Open folder as vault" > select that folder.


================================================================================
  STEP 3: SET UP THE DAILY NOTE TEMPLATE
================================================================================

  1. Inside your vault, create a Templates folder:
       - In Obsidian, click the folder icon (left sidebar) > New Folder
       - Name it: Templates

  2. Create the template file:
       - Inside Templates/, create a new note called "Main Note"
       - Paste this content exactly:

--- START TEMPLATE (copy everything below this line) ---
{{date}} {{time}}

Status:

Tags:

#  Notes



# Meetings




# TODO
- [ ]

# Daily Reflection

#reflection
--- END TEMPLATE (copy everything above this line) ---

  3. Tell Obsidian to use this template:
       - Open Settings (gear icon, bottom-left)
       - Under "Core plugins", make sure "Templates" is toggled ON
       - Under "Core plugins" > "Templates" (click the gear next to it):
           Template folder location:  Templates

  4. (Optional) Set up Daily Notes for one-click new days:
       - Settings > Core plugins > toggle ON "Daily notes"
       - Click the gear next to "Daily notes" and configure:
           Date format:              YYYY-MM-DD
           New file location:        Farther/{{date:YYYY}}/{{date:MMMM}}
           Template file location:   Templates/Main Note

     Now clicking the calendar icon in the left sidebar creates today's
     note pre-filled with your template, saved to:
       Farther/2026/February/2026-02-24.md

  5. Create the folder structure for this year:
       - In Obsidian, create a folder called "Farther" at the vault root
       - Inside Farther/, create "2026"
       - Inside 2026/, create month folders as needed (e.g. "February")

     Or from terminal:
       VAULT="<your-vault-path>"
       mkdir -p "$VAULT/Farther/2026"/{January,February,March,April,May,June,July,August,September,October,November,December}


================================================================================
  STEP 4: DAILY WORKFLOW
================================================================================

  Each morning:
    1. Open Obsidian
    2. Click the calendar icon in the left sidebar (or Cmd+P > "Daily notes: Open today's daily note")
    3. Your note is created with the template pre-filled
    4. Throughout the day:
       - Jot bullet points under "# Notes"
       - Log meetings under "# Meetings"
       - Add tasks under "# TODO" using checkbox syntax:
           - [ ] Send the quarterly report
           - [ ] Review PR for auth service
       - Mark tasks done by clicking the checkbox (turns into [x]):
           - [x] Send the quarterly report
    5. End of day: write a short reflection under "# Daily Reflection"


================================================================================
  STEP 5: INSTALL THE OBSIDIAN MCP SERVER
================================================================================

  Prerequisites:
    - Node.js 18+ (check: node --version)
    - If not installed: brew install node

  Installation:
    1. Clone or copy the obsidian-mcp folder to your machine:
         git clone <repo-url> ~/obsidian-mcp
         cd ~/obsidian-mcp
         npm install

    2. Edit the vault path in obsidian-mcp-server.js:
         Open obsidian-mcp-server.js and find the line:
           const VAULT_ROOT = "..."
         Change it to YOUR vault path:

         If you used iCloud (Option A):
           const VAULT_ROOT = "/Users/<your-username>/Library/Mobile Documents/iCloud~md~obsidian/Documents/Notes";

         If you used local (Option B):
           const VAULT_ROOT = "/Users/<your-username>/Documents/ObsidianVault";

    3. Also update the WORKING_DIR if your daily notes live under a
       different top-level folder. By default it's set to "Farther/".
       Change it to whatever top-level folder name you chose in Step 3.5.

    4. Test the server starts:
         npm start
         (It should start without errors. Press Ctrl+C to stop.)


================================================================================
  STEP 6: ADD THE MCP TO CLAUDE CODE
================================================================================

  Add this to your Claude Code MCP config (~/.claude/mcp.json or
  your project's .mcp.json):

  {
    "mcpServers": {
      "obsidian-vault": {
        "command": "node",
        "args": ["/full/path/to/obsidian-mcp/obsidian-mcp-server.js"]
      }
    }
  }

  Replace /full/path/to/ with the actual path where you installed it.
  Restart Claude Code after saving.


================================================================================
  STEP 7: USE THE MCP TO GET YOUR OUTSTANDING TASKS
================================================================================

  Once connected, you can ask Claude things like:

  "What are my incomplete tasks for this week?"
    -> Uses: get_my_actionable_items with timeframe "this_week"

  "Show me all tasks I haven't finished from last month"
    -> Uses: aggregate_tasks_by_period with period "last_month", filter "incomplete"

  "What did I get done last week?"
    -> Uses: aggregate_tasks_by_period with period "last_week", filter "completed"

  "Generate a task rollover summary from last week"
    -> Uses: create_period_rollover_summary

  "Write my Linear tickets and carryover tasks into today's note"
    -> Uses: write_todos_to_today

  The MCP reads your daily notes, parses all - [ ] and - [x] checkboxes,
  and gives you a clear picture of what's still open.


================================================================================
  QUICK REFERENCE: TASK SYNTAX IN YOUR NOTES
================================================================================

  Incomplete task:     - [ ] Review the budget proposal
  Completed task:      - [x] Review the budget proposal
  With timestamp:      - [x] Review the budget proposal ✅ 2026-02-24
  Sub-task:              - [ ] Check page 3 numbers

  The MCP understands all of these formats.


================================================================================
  TROUBLESHOOTING
================================================================================

  "Daily note not creating in the right folder"
    -> Check Settings > Daily notes > New file location matches your
       folder structure (e.g., Farther/{{date:YYYY}}/{{date:MMMM}})

  "MCP can't find my notes"
    -> Verify VAULT_ROOT in obsidian-mcp-server.js points to your vault
    -> Verify WORKING_DIR matches your top-level folder name
    -> Make sure your daily notes follow YYYY-MM-DD.md naming

  "iCloud sync is slow"
    -> This is normal for first sync. Files sync in the background.
    -> On iPhone/iPad, open the Files app > iCloud Drive to trigger sync.
